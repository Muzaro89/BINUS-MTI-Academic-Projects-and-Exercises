# Cloudflow BI Analysis
In this exercise, we were required to do a thorough analysis using BI software (im using Power BI in this case) from a SaaS company when facing rising customer complaints. Im using various graphs made from the data available to easily observe the data to get more insight.
## Pie charts 
I used pie charts for categorizing complaint tickets to see what is the most complaint is about; and for area distribution of complaints to see which area has the most complaints, indicating which area needs more attention for improvement.

## Line graph
For this case im using the line graph toobserve the trend of the datas, and even compare as two datas may be associated with the other such as the sum of CSAT vs. average time of response

## Clustered bar chart
Using this type of visualization is good to see the performance of each agents by comparing the work each agent has done
